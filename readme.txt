Change the header/footer of the front- and backend of WordPress Multisite

== Description ==

This plugin allows you to add your code/text to the headers and footers on all the blogs in the blog network and in the backend of all blogs too.

In addition, it is possible to replace the footer bar text in the backend completely.

You can access the settings under Network Admin -> Settings.

== Installation ==


1. Upload the  entire plugin to the <code>/wp-content/plugins/</code> directory or install it with WordPress plugin search.
2. Activate the plugin through the 'Plugins' menu in WordPress Network Admin
3. Edit the settings in Network Admin -> Settings
